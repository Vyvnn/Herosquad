# Hero Squad

International hero registry

## Languages Used
- HTML
- Bootstrap
- Java
- Postgresql

## Author
Vivian Muthoni

## Setup
- Clone the repository and navigate to the root of the repository
https://github.com/Vyvnn/Herosquad.git
```
git clone 
cd hero-squad
```

### Direct serving
- Run `gradle build`
- Then run `gradle run`
